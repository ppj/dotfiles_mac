
## Setup

1. Clone the repo in the home folder

2. Symlink (force to remove existing files) the required files
   
   `ln -sf ~/dotfiles_mac/zshrc ~/.zshrc` 
   
   and so on
   
### Note

1. Neovim config
   ```shell
   mkdir ~/.config/nvim
   ln -sf ~/dotfiles_mac/init.vim ~/.config/nvim/init.vim
   ```
